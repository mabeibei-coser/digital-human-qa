# 数字人问答（A900 · digital-human-qa）

一个「数字人 + AI 问答」的网页 Demo：页面一侧是会眨眼 / 说话的数字人（几段视频切换状态），另一侧是创业政策智能问答。
**谁在用 / 用在哪**：永升服务为社保局（创业服务中心）准备的方案演示，用于现场汇报展示；规划上线在 `h100.jsai100.com/a900/`。

## 当前版本

第一版定位「视觉惊艳的演示稿」——重在好看、流畅、像成品。问答先走脚本化演示，暂不接真 AI（接口已留好，后续可接）。

## 数字人三种状态

| 状态 | 说明 |
|---|---|
| 待命 | ~5 秒循环，眨眼、头微动 |
| 欢迎介绍 | 进场播一次，配欢迎 / 主题介绍语（5–8s） |
| 常规说话 | 回答时循环，嘴巴动（~8s） |

视频资产放 `public/avatar/`（见该目录说明），形象技术沿用 `EXP100-形象说话实验`。

## 本地运行

```bash
npm install
npm run dev      # 打开终端里给出的本地地址
```

构建：`npm run build`（产物在 `dist/`，生产 base 为 `/a900/`）。

## 技术栈

Vite + React（纯前端静态站）。无后端、无数据库，适合静态部署 + nginx 子路径托管。

## 相关项目

- `EXP100-形象说话实验`：数字人视频形象的技术来源（透明 WebM）。
- `A400-创业诊断`：同样面向创业 / 就业场景的姊妹项目。
