# 欢迎语音怎么自动播放（重要说明）

## 一句话结论
浏览器**禁止**网页"一打开就直接出声"——Chrome / Edge / Safari / 火狐 + 微信内置浏览器**统一的硬规矩**，为了防止网页乱放声音骚扰用户。任何"开页即出声"的说法，背后要么是用户之前点过、要么是高引擎评分 / 特殊设置。（专门做了对抗式调研，所有"反例"都被证伪，来源见文末。）

所以用三条路保证你的欢迎语音体验：

## 路线 1（已做进网页）：第一下交互即出声，无按钮
- 打开页面，数字人欢迎画面**立刻自动播放**（先静音播画面）。
- 你（演示者）在页面上**随便点一下 / 按一下键 / 碰一下屏**，欢迎语音就**从头有声播放**——不用找按钮，那个"点击播放"按钮已删掉。
- 实际呈现时你一进场总会动鼠标点一下，等于"一开口就有声"。

## 路线 2（正式呈现，保证开页即出声、零交互）：用启动器打开
- 双击项目里的 **`演示-有声启动.bat`**，它用"允许有声自动播放"模式全屏打开页面——**打开瞬间就有声播欢迎语，连点都不用点**。
- 大屏演示 / 展厅一体机最稳的方式。
- 技术上：用浏览器官方认可的 `--autoplay-policy=no-user-gesture-required` 启动参数。

## 路线 3（额外保险）：呈现前先在这台电脑上跑几遍
- 用来呈现的那台电脑，提前把页面打开、让欢迎语音（点一下）完整播放几遍。
- Chrome 会记住"这网站常放有声视频"，之后在这台电脑上**自动允许开页出声**。
- 不完全可控（看 Chrome 内部计分），当额外保险。
